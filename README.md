## Features
- Easy setup for Linux and Windows
- Simple structure to customize for any project

## Getting Started

### Prerequisites
- Git
- Bash (for Linux) or PowerShell (for Windows)

### Installation
#### Linux
```bash
bash setup-linux.sh
```

#### Windows
```powershell
./setup-windows.ps1
```
